# textutils

A small collection of pure, dependency-free Python utility functions:

- **string_helpers.py** — slugify, truncate, whitespace normalization, vowel count, smart title-case
- **validators.py** — email/hex-color validation, clamp, boolean parsing
- **collection_helpers.py** — chunking, dedup, most-common, percentage, running total

All functions are pure (deterministic, no I/O, standard library only).
